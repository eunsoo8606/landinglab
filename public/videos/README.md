# 비디오 파일 안내

이 폴더에 Hero 섹션의 배경 비디오를 추가해주세요.

## 필요한 파일
- `hero-background.mp4` - Hero 섹션 배경 비디오

## 권장 사양
- **해상도**: 1920x1080 (Full HD) 이상
- **형식**: MP4 (H.264 코덱)
- **길이**: 10-30초 (루프 재생됨)
- **파일 크기**: 5MB 이하 (웹 최적화)
- **내용**: 
  - 전문적인 비즈니스 환경
  - 디지털 마케팅 관련 이미지
  - 추상적인 기술/데이터 시각화
  - 사무실 환경 또는 협업 장면

## 무료 비디오 소스
- Pexels: https://www.pexels.com/videos/
- Pixabay: https://pixabay.com/videos/
- Coverr: https://coverr.co/
